# Java Wissensmap

Mein persönliches Lernportal für die Java-Interview-Vorbereitung.  
584 Themen — jeden Tag ein Stück besser.

## Setup

```bash
npm install
npm run dev
```

Öffne dann http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Themen bearbeiten

Jede Seite liegt unter `docs/topics/<kategorie>/<thema>.md`.  
Einfach die Markdown-Datei öffnen und deine Notizen eintragen.
